# Azure 入門

## Azureとは

Microsoft Azureはマイクロソフトが提供するクラウドコンピューティングサービスです。

## 主要サービス

- Azure Virtual Machines
- Azure App Service
- Azure Storage
- Azure SQL Database
- Azure Functions

## 基本的な概念

- リソースグループ
- サブスクリプション
- リージョン
- 可用性セット
